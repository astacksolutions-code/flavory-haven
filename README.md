# 🍽️ Flavory Haven - Professional Restaurant Management System

## 📋 Overview
Complete restaurant management system with admin panel, online reservations, menu management, and discount system.

## ✨ Features
- ✅ Professional UI with Animations
- ✅ Online Table Reservation System
- ✅ Admin Dashboard with Analytics
- ✅ Menu Management (Add/Edit/Delete)
- ✅ Contact Form Management
- ✅ Discount/Coupon System
- ✅ Booking Management (Confirm/Cancel/Delete)
- ✅ MongoDB Atlas Cloud Database
- ✅ Responsive Design (Mobile/Tablet/Desktop)
- ✅ RESTful API Architecture

## 🛠️ Tech Stack
- **Frontend:** Next.js 14, TypeScript, Tailwind CSS, Framer Motion
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Authentication:** JWT

### Prerequisites
- Node.js (v18+)
- MongoDB Atlas Account
